# Profile Screenshots

Save your 3 profile screenshots in this folder with these exact names:

1. **github-profile.png** - GitHub profile screenshot
2. **leetcode-profile.png** - LeetCode profile screenshot  
3. **linkedin-profile.png** - LinkedIn profile screenshot

## Image Requirements:
- **Size**: 1200px × 675px (16:9 aspect ratio) recommended
- **Format**: PNG or JPG
- **File Size**: Keep under 500KB for fast loading
- **Content**: Screenshot of your actual profile pages

## What to Screenshot:
- **GitHub**: Your main profile page showing repositories and contributions
- **LeetCode**: Your profile page showing problem solving stats
- **LinkedIn**: Your professional profile page

Once you save the images with the correct names, they will automatically appear as backgrounds in your profile cards!
